Toolbars

MDC Toolbar acts as a container for multiple rows containing items such as application title, navigation menu, and tabs, among other things. Toolbars scroll with content by default, but supports fixed behavior as well.

Google Material Components documentation

Usage

Directives

mdc-toolbar-row
mdc-toolbar-section
mdc-toolbar-title
mdc-toolbar-fixed-adjust

API reference

Attribute 
@Input() fixed: boolean
 Makes toolbar fixed on top and have persistent elavation. 
@Input() waterfall: boolean
 Removes fixed toolbar persistent elavation; gaining elevation scrolling down the page. 
@Input() flexible: boolean
 Makes toolbar first row has flexible space. 
@Input() flexibleTitle: boolean
 Toolbar text has flexible movement. 
@Input() fixedLastrow: boolean
 Makes only last row of fixed toolbar anchored on top.
Event 
@Output() change
Emits the ratio of current flexible space to total flexible space height.

Examples

menu Toolbar Title file_download print bookmark 
<mdc-toolbar
  [flexible]="false"
  [fixed]="false"
  [waterfall]="false"
  [fixedLastrow]="false"
  [flexibleTitle]="true">
  <mdc-toolbar-row>
    <mdc-toolbar-section [alignStart]="true">
      <a href="#" class="material-icons mdc-toolbar__icon--menu">menu</a>
      <mdc-toolbar-title>Title</mdc-toolbar-title>
    </mdc-toolbar-section>
    <mdc-toolbar-section [alignEnd]="true">
      <a href="#" class="material-icons mdc-toolbar__icon" alt="Download">file_download</a>
      <a href="#" class="material-icons mdc-toolbar__icon" alt="Print page">print</a>
      <a href="#" class="material-icons mdc-toolbar__icon" alt="Bookmark page">bookmark</a>
    </mdc-toolbar-section>
  </mdc-toolbar-row>
</mdc-toolbar>
<main mdc-toolbar-fixed-adjust>
  <p>
    All your page content here will have a default margin-top by using the mdc-toolbar-fixed-adjust directive.
  </p>
</main>
